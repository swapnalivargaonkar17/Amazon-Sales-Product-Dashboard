# Amazon-Sales-Product-Dashboard Power-Bi
This project presents an interactive Power BI dashboard analyzing Amazon product sales performance across multiple dimensions such as time, category, and product-level insights.

# Overview : 
The dashboard provides a comprehensive view of:
 1 Year-to-Date (YTD) Sales: $2.18M 
 2 Quarter-to-Date (QTD) Sales: $811.09K 
 3 YTD Products Sold: 27.75K 
 4 YTD Reviews: 19.42M 
 
It helps stakeholders identify trends, top-performing categories, and high-demand products.
# Key Features :
# Sales Analysis
  Monthly sales trend visualization 
  Weekly sales performance breakdown
  
# Category Insights
Sales contribution by product category 
% share of total YTD sales 
QTD vs YTD comparison 

# Top Performers
Top 5 products by sales 
Top 5 products by reviews 

# Filters
